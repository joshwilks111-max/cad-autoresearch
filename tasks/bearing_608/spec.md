# bearing_608

Real part: the envelope of a standard **608 bearing** (the classic skate/608ZZ size),
sourced as a public STEP. A simple **annular ring** — model the outer cylinder minus
the bore.

- Outer diameter **Ø22 mm** (radius 11)
- Bore diameter **Ø8 mm** (radius 4)
- Width / thickness **7 mm**
- Axis = Z; single solid; no chamfers in this envelope edition.

Spec track. This is a low-face real part (4 faces, euler 2) on the rotationally-symmetric
(cylindrical-IoU) path — the same tractable class as NIST FTC-11.
